Habilidades

Desenvolvimento Front-end

Tenho experiência sólida em HTML, CSS e JavaScript, criando interfaces responsivas e interativas para diferentes plataformas. Uso pré-processadores como SASS/SCSS para escrever CSS de maneira mais eficiente e organizada. Além disso, sou proficiente no uso de frameworks modernos como React.js e Vue.js, e no desenvolvimento de aplicações web utilizando o conceito de componentes reutilizáveis. Em projetos mais complexos, também trabalho com ferramentas de build como Webpack e Babel, e aplico metodologias ágeis, como Scrum, para garantir entregas rápidas e de alta qualidade.

Desenvolvimento Back-end

Tenho um forte domínio em Node.js e PHP, com experiência em criação de APIs RESTful escaláveis​​e seguras. Utilizo frameworks como Express.js (para Node.js) e Laravel (para PHP), implementando autenticação robusta com JWT e OAuth, além de integrar bancos de dados relacionais e não-relacionais, como MySQL, PostgreSQL e MongoDB. Sou especialista em implementar boas práticas de segurança, como proteção contra SQL Injection, XSS, CSRF e uso de criptografia adequada para senhas e dados sensíveis Minha abordagem envolve desenvolvimento orientado a testes (TDD) e versionamento de código com Git.

Design de UX/UI

Tenho um foco constante na experiência do usuário, desenvolvendo interfaces intuitivas e acessíveis. Utilize ferramentas como Figma, Sketch e Adobe XD para prototipar e criar layouts que atendam às necessidades dos usuários. Sou capaz de conduzir pesquisas de usuário (como entrevistas, testes de usabilidade e análise de dados de comportamento), para garantir que as soluções atendam às expectativas e proporcionem uma navegação fluida. Também aplico design responsivo e princípios de design inclusivo, garantindo que as interfaces funcionem bem em diversos dispositivos e sejam acessíveis a todos os usuários, incluindo aqueles com deficiência.
